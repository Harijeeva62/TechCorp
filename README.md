# TechCorp Corporate Website

This project is a modern responsive corporate website developed as part of the **Rytsense Round 1 Technical Assessment**. The application demonstrates front-end development skills, UI design, and responsive layout implementation using React and modern web technologies.

## Features
- Responsive corporate website design
- Home page with hero section and company introduction
- About page with company vision, mission, and values
- Services page showcasing technology solutions
- Contact page with contact form and company details
- Dark and light theme support
- Animated UI components using Framer Motion
- Interactive map integration
- Contact form with Supabase database integration

## Technology Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- React Router DOM
- Framer Motion
- Supabase
- Leaflet (OpenStreetMap)

## Project Setup Instructions

1. Clone the repository
   
2. Install the required dependencies
   **npm i --force**
3. Build the entire project
   **npm run build**
4. Run the development server
   **npm run dev**
